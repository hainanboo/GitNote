### 内容

**README**的内容通常包含：

- **设置指南**
- **安装说明**
- **操作指南**
- **档案列表（随这软件而发布的档案）**
- **版权声明**
- **软件作者的联络方法**
- **已知程序的错误**
- **疑难排解**
- **致谢**
- **更新记录**



### 通用界面说明

**README**偶尔也会被用来形容一些提供资讯的档案，包含**README**普遍的内容。在一些开放源代码的程式，特别是以[GNU](https://zh.wikipedia.org/wiki/GNU)建置系统打包的档案里，通常会包含：

- **README**：一般资讯
- **AUTHORS**：作者
- **THANKS**：致谢
- **ChangeLog**：更新记录，通常为程序管理员而设
- **NEWS**：更新记录，但专为使用者而设
- **INSTALL**：安装指南
- **COPYING**：版权声明
- **BUGS**：已知程序错误，亦包括汇报错误的方法
- **FAQ**：有关软件的回答
- **TODO**：更新预告



### 说明事项

1. 软件定位，软件的基本功能
2. 运行代码的方法：安装环境，启动命令等
3. 简要的使用说明
4. 代码目录结构说明，更详细店可以说明软件的基本原理
5. 常见问题说明
